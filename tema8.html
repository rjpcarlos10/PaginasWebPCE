Tema 8: Selectors

Definición

Los selectores en CSS indican qué elementos HTML serán afectados por un conjunto de reglas. Existen diferentes tipos de selectores, entre ellos: de etiqueta, de clase, de ID, universales, de atributo, pseudoclases, entre otros.

Tipos comunes de selectores

  • Selector de etiqueta: Aplica el estilo a todas las etiquetas de ese tipo. p { color: blue; }
  • Selector de clase: Aplica el estilo a todos los elementos que tienen esa clase. .nombreDeClase { color: red; }
  • Selector de ID: Aplica el estilo al elemento que tenga ese ID único. #nombreDeID { color: green; }
  • Selector universal: Aplica a todos los elementos. * { margin: 0; }
  • Selector de atributo: Aplica a los elementos con un atributo específico. a[target="_blank"] { color: pink; }

Ejemplo de uso

<!-- HTML -->
<div class="contenedor">
    <p id="textoPrincipal">Hola Mundo</p>
    <p>Este es otro párrafo</p>
</div>

<!-- CSS -->
/* Selector de etiqueta */
p {
    color: blue;
}

/* Selector de ID */
#textoPrincipal {
    font-weight: bold;
}

/* Selector de clase */
.contenedor {
    border: 2px solid #333;
    padding: 10px;
}
