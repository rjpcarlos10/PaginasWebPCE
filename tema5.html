Tema 5: CSS Font

Definición

Con las propiedades de fuente en CSS podemos cambiar la tipografía, tamaño, grosor y estilo de texto en un sitio web.

Sintaxis

selector {
    font-family: Arial, sans-serif;
    font-size: 16px;
    font-weight: bold;    /* normal, bold, bolder, lighter, o numéricos (100,200,300...) */
    font-style: italic;   /* normal, italic, oblique */
}

Uso

Con font-family especificamos la tipografía principal y fuentes alternativas. font-size determina el tamaño de la fuente, y font-weight define su grosor.

Ejemplo

p {
    font-family: 'Helvetica', sans-serif;
    font-size: 14px;
    font-weight: normal;
    font-style: italic;
}
