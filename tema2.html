Tema 2: Sintaxis básica de funcionalidades CSS

Definición

CSS (Cascading Style Sheets) se utiliza para dar estilo y formatear la apariencia de las páginas web. La sintaxis de CSS se basa en selectores y declaraciones que definen cómo se verán los elementos HTML.

Sintaxis básica

selector {
    propiedad: valor;
    propiedad: valor;
    ...
}

Uso

Se declara un selector (un elemento HTML, una clase o un id) seguido de llaves { }. Dentro de ellas se ponen las propiedades y valores que afectarán el estilo de dicho elemento, por ejemplo, color de fondo, fuentes, márgenes, bordes, etc.

Ejemplo

/* Ejemplo de un estilo básico en CSS */
body {
    background-color: #f5f5f5;
    font-family: Arial, sans-serif;
}
p {
    color: #333333;
    line-height: 1.5;
}
