Tema 6: HTML Styles - CSS

Definición

La etiqueta <style> en HTML permite incluir reglas CSS directamente dentro del documento. También es posible usar el atributo style en línea para aplicar estilos a elementos individuales, aunque no es lo más recomendable para proyectos grandes.

Sintaxis y uso

Existen 3 formas de aplicar CSS a un documento HTML:

  1. CSS en línea: Utilizando el atributo style directamente en la etiqueta HTML.
  2. CSS interno: Mediante la etiqueta <style> en la sección <head>.
  3. CSS externo: En un archivo independiente (.css) enlazado con la etiqueta <link> dentro de <head>.

Ejemplo

<!-- Ejemplo de CSS en línea -->
<p style="color:red; font-size:20px;">Este párrafo está en rojo</p>

<!-- Ejemplo de CSS interno -->
<style>
  p {
    color: blue;
    font-size: 18px;
  }
</style>

<!-- Ejemplo de CSS externo -->
<link rel="stylesheet" href="estilos.css" />
