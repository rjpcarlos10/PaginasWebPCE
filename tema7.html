Tema 7: CSS Syntax - CSS

Definición

La sintaxis de CSS se basa en reglas compuestas por un selector y declaraciones entre llaves. Cada declaración contiene una propiedad y un valor.

Sintaxis

selector {
    propiedad: valor;
    propiedad: valor;
    ...
}

Uso

El selector determina a qué elemento(s) HTML se aplicarán los estilos. Las propiedades y valores aplican el aspecto deseado.

Ejemplo

h2 {
    color: #0f4c81;
    font-size: 24px;
    font-weight: bold;
}
