Tema 1: Creación de un portal básico en HTML

Definición

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear la estructura de una página web. Un portal básico en HTML incluye etiquetas fundamentales que definen la página y su contenido.

Sintaxis básica

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Título de la página</title>
</head>
<body>

    <!-- Aquí va el contenido de la página -->

</body>
</html>

Uso

Esta estructura define la columna vertebral de cualquier documento HTML. La etiqueta <!DOCTYPE html> indica al navegador la versión de HTML. <head> contiene metadatos, título, enlaces a hojas de estilo, etc., mientras que <body> contiene todo el contenido visible en la página.

Ejemplo

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Mi Primer Portal HTML</title>
</head>
<body>
    <h1>Bienvenidos a mi Portal</h1>
    <p>Este es un sitio web sencillo creado con HTML.</p>
</body>
</html>
